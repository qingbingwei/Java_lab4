# 项目部署说明

## 环境要求

- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

## 启动步骤

### 1. 进入前端目录
```bash
cd front_end
```

### 2. 安装依赖（首次运行）
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

系统会自动在浏览器中打开 http://localhost:5173

### 4. 使用系统
首次使用时，请点击页面右上角的"初始化数据"按钮，系统将自动生成：
- 120个学生
- 8个教师
- 6门课程
- 若干教学班
- 完整的成绩数据

## 功能模块

1. **首页概览** - 查看系统统计数据和快捷入口
2. **学生管理** - 浏览和搜索学生信息
3. **教师管理** - 查看教师授课情况
4. **课程管理** - 查看课程信息
5. **教学班管理** - 查看班级和选课情况
6. **成绩录入** - 分步骤录入各类成绩
7. **成绩查询** - 多条件查询学生成绩
8. **成绩排名** - 查看学生排名
9. **统计分析** - 查看图表化的统计数据
10. **选课管理** - 查看选课统计

## 特色功能

✨ **无需后端** - 使用Mock数据，独立运行
✨ **数据持久化** - 数据保存在浏览器本地
✨ **主题切换** - 支持深色/浅色模式
✨ **响应式设计** - 适配PC和移动端
✨ **数据可视化** - ECharts图表展示
✨ **现代化UI** - 基于Element Plus

## 技术栈

- Vue 3 (Composition API)
- Vite
- Element Plus
- Axios
- Pinia
- ECharts
- Sass

## 注意事项

⚠️ 数据存储在浏览器localStorage中，清除缓存会导致数据丢失
⚠️ 建议使用Chrome、Firefox、Edge等现代浏览器
⚠️ 首次使用必须先初始化数据

## 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录

## 预览生产版本

```bash
npm run preview
```

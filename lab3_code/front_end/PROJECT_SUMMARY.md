# 学生成绩管理系统 - 前端项目总结

## 📋 项目概述

本项目是基于实验1后端业务模型开发的前后端分离式系统的前端界面，采用 **Vue3 + Axios + Element Plus** 技术栈，实现了完整的学生成绩管理功能。

**开发状态：✅ 已完成并成功运行**
**访问地址：http://localhost:5173/**

## 📊 数据模型

基于实验1的业务模型：

### 学生 (Student)
```javascript
{
  studentId: String,        // 学号
  name: String,            // 姓名
  gender: String,          // 性别 (MALE/FEMALE)
  enrolledClasses: Array,  // 已选课程列表
  scores: Object          // 成绩记录
}
```

### 教师 (Teacher)
```javascript
{
  teacherId: String,       // 教师编号
  name: String,           // 姓名
  teachingClasses: Array  // 授课班级
}
```

### 课程 (Course)
```javascript
{
  courseId: String,       // 课程编号
  courseName: String,     // 课程名称
  credits: Number        // 学分
}
```

### 教学班 (TeachingClass)
```javascript
{
  classId: String,        // 班级编号
  course: Object,         // 课程信息
  teacher: Object,        // 教师信息
  students: Array,        // 学生列表
  capacity: Number,       // 容量
  semester: String       // 学期
}
```

### 成绩 (Score)
```javascript
{
  classId: String,            // 班级编号
  regularScore: Number,       // 平时成绩 (20%)
  midtermScore: Number,       // 期中成绩 (20%)
  experimentScore: Number,    // 实验成绩 (20%)
  finalExamScore: Number,     // 期末成绩 (40%)
  finalScore: Number         // 综合成绩
}
```

**综合成绩计算公式**：
```
finalScore = regularScore × 0.2 + midtermScore × 0.2 + 
             experimentScore × 0.2 + finalExamScore × 0.4
```

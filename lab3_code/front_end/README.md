# 学生成绩管理系统 - 前端

基于 Vue 3 + Vite + Element Plus + Axios 开发的学生成绩管理系统前端界面。

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 下一代前端构建工具
- **Element Plus** - 基于Vue 3的组件库
- **Axios** - HTTP客户端
- **Vue Router** - 官方路由管理器
- **Pinia** - Vue状态管理
- **ECharts** - 数据可视化图表库

## 功能特点

### 核心功能
1. **数据管理**
   - 学生信息管理（查看、搜索）
   - 教师信息管理
   - 课程信息管理
   - 教学班管理

2. **成绩管理**
   - 成绩录入（平时、期中、实验、期末）
   - 成绩查询（按学生、按课程）
   - 成绩统计分析
   - 成绩排名展示

3. **数据可视化**
   - 成绩分布图表
   - 学生成绩趋势分析
   - 课程成绩对比
   - 统计数据概览

### 扩展特色功能
1. **智能搜索** - 支持多条件组合搜索
2. **数据导出** - 支持导出Excel格式
3. **响应式设计** - 适配不同屏幕尺寸
4. **主题切换** - 支持浅色/深色主题
5. **操作历史** - 记录重要操作日志
6. **批量操作** - 支持批量录入成绩
7. **数据缓存** - 优化加载性能
8. **友好提示** - 完善的用户操作反馈

## 项目结构

```
front_end/
├── public/              # 静态资源
├── src/
│   ├── api/            # API接口定义
│   ├── assets/         # 资源文件（图片、样式等）
│   ├── components/     # 公共组件
│   ├── layouts/        # 布局组件
│   ├── router/         # 路由配置
│   ├── stores/         # Pinia状态管理
│   ├── utils/          # 工具函数
│   ├── views/          # 页面视图

│   └── main.js         # 入口文件
├── index.html          # HTML模板
├── vite.config.js      # Vite配置
└── package.json        # 项目配置
```

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 使用说明

### 1. 数据初始化
首次使用时，需要初始化系统数据，生成学生、教师、课程和教学班信息。

### 2. 选课管理
为学生分配课程，每个学生至少选择3门课程。

### 3. 成绩录入
按阶段录入成绩：平时成绩 → 期中成绩 → 实验成绩 → 期末成绩

### 4. 成绩查询
支持按学号、姓名、课程等条件查询学生成绩。

### 5. 数据统计
查看成绩分布、平均分、及格率等统计数据。

## 注意事项

- 本项目使用Mock数据模拟后端接口，不需要实际的Java后端服务
- 所有数据存储在浏览器本地存储中
- 建议使用Chrome、Firefox、Edge等现代浏览器

## 界面预览

- 主页：系统概览和快捷入口
- 学生管理：学生信息列表和详情
- 成绩管理：成绩录入和查询界面
- 数据统计：图表展示成绩统计
- 排名查看：学生成绩排名列表

# 学生成绩管理系统前端 - 快速启动指南

## 项目说明

这是一个基于 **Vue 3 + Vite + Element Plus + Axios + ECharts** 开发的学生成绩管理系统前端项目。

## 功能特色

### ✨ 核心功能
- 📊 **首页概览** - 系统数据统计、快捷入口、功能说明
- 👨‍🎓 **学生管理** - 学生信息查看、搜索筛选、详情展示
- 👨‍🏫 **教师管理** - 教师信息、授课情况
- 📚 **课程管理** - 课程列表、选课统计
- 🏫 **教学班管理** - 班级信息、容量管理
- ✏️ **成绩录入** - 分步骤录入各类成绩
- 🔍 **成绩查询** - 多条件组合查询
- 🏆 **成绩排名** - 多种排名方式
- 📈 **统计分析** - ECharts可视化图表
- 📋 **选课管理** - 选课情况查看

### 🎨 界面特色
- 现代化设计风格
- 响应式布局，支持移动端
- 深色/浅色主题切换
- 流畅的动画效果
- 丰富的图表展示

### 🚀 技术亮点
- **Vue 3 Composition API** - 最新的组合式API
- **Vite** - 极速的开发体验
- **Element Plus** - 完整的组件库
- **Pinia** - 轻量级状态管理
- **ECharts** - 专业的数据可视化
- **Mock数据** - 无需后端即可运行
- **本地存储** - 数据持久化

## 安装步骤

### 1. 进入项目目录
```bash
cd front_end
```

### 2. 安装依赖
```bash
npm install
```

如果安装速度慢，可以使用国内镜像：
```bash
npm install --registry=https://registry.npmmirror.com
```

### 3. 启动开发服务器
```bash
npm run dev
```

启动成功后，浏览器会自动打开 `http://localhost:5173`

### 4. 构建生产版本（可选）
```bash
npm run build
```

### 5. 预览生产构建（可选）
```bash
npm run preview
```

## 使用说明

### 首次使用

1. **初始化数据**
   - 点击右上角的"初始化数据"按钮
   - 系统会自动生成120个学生、8个教师、6门课程和若干教学班
   - 同时会生成所有学生的成绩数据

2. **浏览系统**
   - 使用左侧菜单导航到不同功能模块
   - 首页展示了系统概览和快捷入口

### 主要功能使用

#### 📊 学生管理
- 查看所有学生列表
- 搜索学号或姓名
- 按性别筛选
- 点击"详情"查看学生完整信息和成绩

#### ✏️ 成绩录入
1. 选择教学班
2. 选择成绩类型（平时/期中/实验/期末）
3. 录入成绩（可批量生成）
4. 提交保存

#### 🔍 成绩查询
- 支持按学号、姓名、课程查询
- 可筛选成绩范围
- 查看详细的成绩信息

#### 🏆 成绩排名
- 综合成绩总分排名
- 平均成绩排名
- 按课程排名
- 前三名会有奖牌标识

#### 📈 统计分析
- 查看整体统计数据
- 成绩分布饼图
- 课程平均分柱状图
- 成绩段分布雷达图
- 及格率对比折线图
- 课程详细统计表格

### 数据管理

- **数据持久化**：所有数据自动保存到浏览器本地存储
- **重新初始化**：随时可以重新初始化生成新数据
- **数据导出**：支持导出Excel（界面已实现，可自行对接真实导出功能）

### 主题切换

点击顶部导航栏的主题切换开关，可在浅色和深色主题间切换。

## 注意事项

1. **无需后端**：本项目使用Mock数据，完全独立运行
2. **数据存储**：数据保存在浏览器localStorage中
3. **初始化数据**：首次使用需要点击"初始化数据"按钮
4. **清除数据**：清除浏览器缓存会导致数据丢失
5. **端口占用**：如果5173端口被占用，Vite会自动使用下一个可用端口

## 常见问题

### Q: 无法安装依赖？
A: 检查Node.js版本（建议16+），尝试使用国内镜像源。

### Q: 启动后页面空白？
A: 检查浏览器控制台错误信息，确保依赖正确安装。

### Q: 数据丢失了？
A: 数据存储在localStorage中，清除浏览器缓存会导致数据丢失。重新初始化即可。

### Q: 如何对接真实后端？
A: 在 `src/api/` 目录下添加真实的API接口，修改 `vite.config.js` 中的代理配置。
